About Dreams Come True

This is the original demo track for “All Falls Down”, from Kanye’s album The College Dropout. The song is essentially the same, but features some different lyrics and a different beat. It seems as if, after revisions were made, Yeezy removed the drug references (“Cut that coke”, “Bricks, nicks, nigga so herb”).
